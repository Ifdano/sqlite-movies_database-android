## Тестовое приложение для практики, демонстрирующее работу с базой данных SQLite.

### Подробнее.

### 1] Главный экран, на котором отображены названия фильмов, взятые из базы данных. Список прокручиваемый.
![project_movielist_main_ver'](https://user-images.githubusercontent.com/15383481/84548998-fa18be00-ad17-11ea-82d7-4a74a988bcbb.png)
![project_movielist_main_hor'](https://user-images.githubusercontent.com/15383481/84548992-f7b66400-ad17-11ea-9f3f-6ddfa248c66e.png)
### Удалить фильм можно продолжительным нажатием на сам фильм. Будет выведено диалоговое окно о подтверждении удаления. Фильм будет удален из базы данных и из списка. 
### Для получения подробной информации о фильме - нужно выбрать его из списка.

### 2] Экран, на котором можно добавить новый фильм в базу данных. Будет выведено диалоговое окно о подтверждении добавления фильма в базу данных.
![project_movielist_add_ver'](https://user-images.githubusercontent.com/15383481/84549227-862ae580-ad18-11ea-8047-52a20d6f709d.png)
![project_movielist_add_hor'](https://user-images.githubusercontent.com/15383481/84549220-83c88b80-ad18-11ea-8bfe-3813f1f2138c.png)

### 3] Экран, на котором можно получить подробную информацию о выбранном фильме. Выводится вся доступная информация.
![project_movielist_info_ver'](https://user-images.githubusercontent.com/15383481/84549426-ff2a3d00-ad18-11ea-8584-2afea5f39b65.png)
![project_movielist_info_hor'](https://user-images.githubusercontent.com/15383481/84549424-fdf91000-ad18-11ea-960f-38874b1f8e21.png)
### Можно сразу удалить фильм. Фильм будет удален из базы данных и из списка. Будет выведено диалоговое окно о подтверждении удаления.
### Можно обновить информацию о фильме, перейдя на экран с "редактором".

### 4] Экран, на котором можно обновить информацию о фильме. Будет сделан запрос в базу данных для обновления информации.
![project_movielist_update_ver'](https://user-images.githubusercontent.com/15383481/84549627-7eb80c00-ad19-11ea-9219-3f15bbed49ff.png)
![project_movielist_update_hor'](https://user-images.githubusercontent.com/15383481/84549624-7d86df00-ad19-11ea-8666-e85ccace937f.png)

### Примечание:

#### 1] Это лишь простой пример для практики. 
#### 2] Почти все окна содержат фрагменты.
#### 3] Для облегчения подстройки под разные разрешения экрана, были использованы единицы sdp/ssp.

#### sdp: https://github.com/intuit/sdp
#### ssp: https://github.com/intuit/ssp
